## 🧠 &nbsp;Agent Design & Key Features

<div align="center">

<table>
<tr>
<td width="50%" align="center">

### 🤖 Multi-Agent Planning & Healing

| State Node | Function |
|------------|----------|
| 🟢 **Classify** | Discovers intent (direct chat vs task flow) |
| 📋 **Plan** | Creates a graph of execution steps |
| 🚀 **Execute** | Invokes Registry tools securely |
| 🩹 **Heal** | Captures errors and plans self-fixes |
| 🔄 **Replan** | Reformulates goals on blockages |

</td>
<td width="50%" align="center">

### 🎙️ Low-Latency Voice Stack

<div style="border: 1px solid var(--border); border-radius: 4px; padding: 10px; background: rgba(0,0,0,0.1); width: 90%; text-align: left;">

- **Mic Calibration**: Energy-based noise filters.
- **Always-On Wake Words**: Lightweight ONNX models for phrase detection.
- **Offline Transcription**: Whisper-based local speech recognition.
- **Responsive TTS**: Word-by-word streaming synthesiser matching agent voice replies.

</div>

</td>
</tr>
</table>
